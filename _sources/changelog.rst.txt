[0.1.1] - 2025-01-XX
---------------------

Added
~~~~~
- Initial release of SemanticLens
- Core Lens class for semantic analysis
- Component visualization strategies (activation-based, relevance-based)
- Foundation model integration (OpenCLIP)
- Interpretability metrics (clarity, polysemanticity, redundancy, similarity)
- Text probing functionality
- Comprehensive test suite
- Documentation and tutorials

Features
~~~~~~~~
- Support for multiple component visualization strategies
- Integration with popular foundation models
- Quantitative interpretability metrics
- Natural language probing of model internals
- Extensible architecture for custom visualizers and foundation models
